## 📝 프로젝트 개요
> **프로젝트 :** 하양 맛집 소개 사이트 <br/><br/>
**프로젝트 소개:**대학가 맛집 정보를 얻고 싶었지만 지방이라 학교 커뮤니티를 검색하거나 네이버 지도에서 하나하나 찾아봐야 했습니다. 대학 근처 맛집을 모아 한 번에 볼 수 있는 사이트가 있으면 좋겠다는 생각에 제작하게 되었습니다.<br/><br/>
**분류:** 개인 프로젝트  <br/><br/>
