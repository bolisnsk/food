나의 첫 학교에서 시켜서 하는 것이 아닌 개인 프로젝트!

학교 주변을 중심으로 한식, 중식, 양식, 카페 및 디저트로 나누어서 맛집을 소개하는 사이트다.

메인 화면은 그리드로 구성하여 각 음식집의 사진을 삽입하였다. 터치하거나 마우스를 갖다대면 음식점 이름이 나온다.
누르면 그 음식집에 대한 정보를 보여주는 페이지로 이동한다.
그리드 하단에 더보기를 눌러 각 분야별 음식집 리스트를 볼 수 있다. 마찬가지로 원하는 음식집의 이름을 누르면 정보를 보여주는 페이지로 이동한다.

정보 페이지에는 음식집 이름, 위치, 전화번호 등을 넣을 것이며 api를 이용하여 지도도 불러올 생각이다.

댓글창은...거기까진 구현 못할 것 같고 사이트에서 무료로 제공하는 것을 가져올 것 같다. (삽입한다면 말이다.)
